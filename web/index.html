Getting Started

Ready to use: The Agent Coding Assistant is configured and ready to answer your questions.
1Choose Your Session
  1. Each conversation is tied to a unique Session ID
  2. You can create a new session or continue an existing one
  3. Use the + button to increment to a new session
  4. Previous sessions can be accessed via the history button
2Select AI Complexity
  1. Easy: Quick answers to simple Google Ads questions
  2. Medium: Detailed analysis and strategic recommendations
  3. Hard: Complex problem-solving and multi-faceted strategies
3Start Asking Questions
Example queries to try:
  • "How do I set up a Performance Max campaign?"
  • "What's the best bidding strategy for e-commerce?"
  • "How can I improve my Quality Score?"
  • "Explain different Google Ads campaign types"
  • "What are responsive search ads best practices?"

Pro Tips
